# ACS 1320 Break Out

## Run Locally

### Clone the project

```bash
  git clone https://github.com/alexcrocha/ACS-1320-break-out
```

### Go to the project directory

```bash
  cd ACS-1320-break-out
```

### Install dependencies

```bash
  npm install
```

### Start the game

MacOS/Linux

```bash
  npm run start
```

Windows

```bash
  npm run start-win
```
